## Descrição
Sistema em Python para cadastro de alunos, com persistência em arquivo `.csv` utilizando pandas.

Funcionalidades:
- Inserir aluno
- Pesquisar por matrícula ou nome (case-insensitive)
- Editar dados do aluno (exceto matrícula)
- Remover aluno
- Menu em loop até opção sair
